## 方法一：网页版使用（推荐）

### 步骤 1：部署到 Vercel

在电脑上执行以下操作：

```bash
# 1. 进入项目目录
cd /Users/tonysheng/Desktop/编程/高考复习

# 2. 安装 Vercel CLI（如果没有）
npm install -g vercel

# 3. 登录 Vercel
vercel login

# 4. 部署项目
vercel --prod
```

### 步骤 2：配置环境变量

在 Vercel 控制台设置以下环境变量：

| 变量名 | 值 | 说明 |
|--------|-----|------|
| `GEMINI_API_KEY` | 你的 Gemini API Key | 从 [Google AI Studio](https://aistudio.google.com/) 获取 |
| `GETNOTES_TOKEN` | 你的 Get笔记 Token | 从 Get笔记开发者设置获取 |
| `JWT_SECRET` | 随机字符串 | 用于加密用户登录状态 |

### 步骤 3：在手机上使用

1. 打开手机浏览器（推荐 Chrome）
2. 访问你的 Vercel 部署地址，例如：`https://gaokao-xxx.vercel.app`
3. **添加到主屏幕**（可选，像 APP 一样使用）：
   - Chrome：点击右上角菜单 → "添加到主屏幕"
   - Safari：点击分享按钮 → "添加到主屏幕"

## 方法二：安卓APP安装

### 准备工作

确保电脑已安装：
- Node.js 18+
- npm 或 yarn

### 步骤 1：安装依赖

```bash
# 进入移动端目录
cd /Users/tonysheng/Desktop/编程/高考复习/高考复习-mobile

# 安装依赖
npm install
```

### 步骤 2：配置 API 地址

编辑 `constants/api.ts` 文件：

```typescript
// 将 your-vercel-app 替换为你的实际部署地址
export const API_BASE_URL = __DEV__ 
  ? 'http://10.0.2.2:3000/api'
  : 'https://你的实际部署地址.vercel.app/api';
```

### 步骤 3：构建 APK

**方法 A：使用 EAS Build（推荐）**

```bash
# 安装 EAS CLI
npm install -g eas-cli

# 登录 Expo 账号（没有就注册一个）
eas login

# 配置项目
eas build:configure

# 构建 APK
eas build --platform android --profile preview
```

构建完成后，会提供下载链接。

**方法 B：本地构建**

```bash
# 安装 Expo 开发客户端
npx expo install expo-dev-client

# 运行构建
npx expo run:android
```

### 步骤 4：安装到手机

1. 将 APK 文件传输到手机（通过 USB、网盘、蓝牙等）
2. 在手机上打开文件管理器，找到 APK 文件
3. 点击安装
   - 如提示"未知来源"，去设置中允许安装
4. 安装完成后，在桌面找到"高考冲刺"图标

## 功能使用指南

### 🎓 AI 导师（核心功能）

采用苏格拉底式教学法的智能辅导。

**使用步骤：**
1. 选择学科（数学、物理、化学等）
2. 选择学习模式：
   - 📖 **学习模式**：系统学习新知识点
   - 🔄 **复习模式**：复习薄弱环节
   - 🎯 **测验模式**：做题检验掌握程度
3. 输入知识点或直接提问
4. 与 AI 进行对话式学习

**特点：**
- AI 会先询问你已知的内容
- 基于你的回答逐步深入讲解
- 每次讲解后会提问验证理解
- 自动记录学习进度

### 📊 试题分析

基于你的 Get笔记知识库（历年高考真题）进行分析。

**使用步骤：**
1. 选择要分析的学科
2. 可选输入具体知识点
3. 点击"开始分析预测"
4. 查看分析结果：
   - 🔥 高频考点 TOP 5
   - 📈 命题趋势分析
   - 🎯 今年预测重点
   - 📋 针对性复习建议

### 📈 学习进度

跟踪各知识点的掌握程度。

**查看内容：**
- 总体掌握度百分比
- 各学科进度概览
- 已掌握/学习中/未开始的知识点
- 每个知识点的详细学习记录

### 📷 拍题讲解

拍照上传题目，AI 详细解答。

**使用步骤：**
1. 点击上传区域，拍照或选择图片
2. 选择学科
3. 点击"开始分析题目"
4. 查看 AI 解答
5. 可将题目保存到错题本

### 📚 笔记搜索

搜索 Get笔记知识库中的内容。

**已接入的知识库：**
- `K0BlyZmn` - 高考试题知识库

### 📝 错题本

保存和管理分析过的题目。

**功能：**

- 查看原题和解析
- 删除不需要的记录

## 常见问题

### Q: 注册/登录失败？

A: 
1. 检查网络连接
2. 用户名不要包含特殊字符
3. 密码至少 6 位

### Q: AI 回复很慢？

A:
1. 网络问题，等待或重试
2. API 限额，稍后再试

### Q: 知识库搜索没结果？

A:
1. 尝试更简短的关键词
2. 知识库可能未收录该内容
3. 系统会自动降级到 AI 生成

### Q: APK 安装失败？

A:
1. 开启"允许安装未知来源应用"
2. 确保手机存储空间充足
3. Android 版本需 8.0 以上

### Q: 如何更新APP？

A:
1. 网页版自动更新
2. APK版需重新构建安装
